# Ticket Booking Automation Framework

## 📌 Project Overview
This project automates a Ticket Booking Web Application using Selenium, Pytest, and Page Object Model (POM).

## 🚀 Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Pytest-HTML
- WebDriver Manager

## 📂 Framework Features
- Page Object Model (POM)
- Data-Driven Testing (CSV)
- Logging & Reporting
- Screenshot on Failure
- Configurable Test Execution

## ▶️ How to Run Tests

### Step 1: Install dependencies
pip install -r requirements.txt

### Step 2: Run tests
pytest

### Step 3: Generate report
pytest --html=reports/report.html

## 📊 Test Scenarios Covered
- Valid ticket booking
- Empty field validation
- Same source & destination
- Invalid date scenarios

## 📸 Reporting
- HTML Report generated
- Screenshots captured on failure
